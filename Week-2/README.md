# Backend Project (backend-1)

## Description

A simple backend server built using Node.js and Express.

## Features

* Express server setup
* API routing
* JSON handling
* REST API testing

## Installation

Move into backend folder:

```bash
cd backend-1
Install dependencies:

npm install
Run the server:

node server.js
or

npm start
Server runs on:

http://localhost:3000
