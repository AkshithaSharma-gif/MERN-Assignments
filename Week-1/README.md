========Week 1=========

This repository contains basic JavaScript programs that demonstrate fundamental programming concepts such as arrays, functions, loops, command-line arguments, and simple problem-solving techniques.

week1/ │ ├── arraySumFunctionQues6.js ├── arraySumQues3.js ├── bignumberQues1.js ├── employeeArrayOperationsQues8.js ├── findLargestNumberFromArgsQues5.js ├── largestOfThreeQues2.js ├── searchElementIndexQues7.js └── smallestElementQues4.js

========Program Descriptions=========

bignumberQues1.js
Finds the largest number among given inputs. Demonstrates comparison logic.

largestOfThreeQues2.js
Takes three numbers as input. Prints the largest among them. Uses conditional statements.

arraySumQues3.js
Calculates the sum of all elements in an array. Uses loops for iteration.

smallestElementQues4.js
Finds the smallest element in an array. Uses comparison logic inside loops.

findLargestNumberFromArgsQues5.js
Accepts numbers from command-line arguments. Finds the largest number dynamically.

arraySumFunctionQues6.js
Computes sum of array elements using a function. Demonstrates function reuse and modular coding.

searchElementIndexQues7.js
Searches for an element in an array. Returns index if found, otherwise returns -1. Implements linear search.

employeeArrayOperationsQues8.js
Performs basic operations on an employee array. May include add, display, search, or update operations. Demonstrates array of objects usage.

Arrays and array operations Functions in JavaScript Loops and conditionals Command-line arguments Searching and comparison logic Basic data handling
